About

As time passes, OpenTofu and Terraform become more distant from each other. CanI.TF helps us to understand their differences quickly.

  • How to add a new feature?

    To add a new feature or correct a mistake create a Pull Request.

  • Is this list available in a different format?

    Yes, you can get the raw JSON.

  • Stacks*

    While Terraform did gain a feature named Stacks in version 1.9, it's relevant to note that it's not available as part of the community edition product licensed under BUSL-1.1.

    It's an HCP Terraform product that requires using a HashiCorp hosted service.

    All OpenTofu features are available for use without subscription from a hosted service.

  • What's the licence?

    Creative Commons Attribution 4.0 International.
